# AGENTS.md

## Цель проекта
Разделить монолитный HTML-проект на отдельные данные, стили и JS-модули
без изменения визуального результата и логики интерфейса.

## Правила работы
- Не использовать фреймворки.
- Не подключать сборщики без явной необходимости.
- Делать минимальные и обратимые изменения.
- Сначала рефакторинг структуры, потом улучшения.
- Не менять тексты, числа и смысл контента без необходимости.
- После каждого изменения кратко описывать:
  1. какие файлы изменены
  2. что именно вынесено
  3. что осталось сделать
- Если есть риск сломать поведение, сначала предложить план без правок.

## Definition of done
- index.html содержит только базовый каркас страницы
- данные вынесены в папку data
- стили вынесены в папку styles
- логика вынесена в js-модули
- приложение запускается локально без ошибок
- визуально проект максимально близок к исходной версии
